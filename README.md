# Book Inventory (React + localStorage)

Sistema simples de estoque de livros. Funcionalidades incluídas (conforme a tabela):
- CRUD de tabelas (Books, Movements, Programs simulated via movements)
- CSS puro
- Tabela Movimentação e Programação Movimentação (movements list and scheduling placeholder)
- Validação em formulários
- Página principal e menu
- Autenticação e autorização (login fake)
- Geração de dados em tabelas (seed)
- Filtrar dados em manter (filters in list)
- Consultas com dados em múltiplas tabelas com totais e cálculos (totals of stock & value)
- Consulta de agrupamento (group by genre/author)
- Consulta tipo Pivot (simple pivot counts)
- Uso de Git: commit-ready project

## Como usar
1. `npm install`
2. `npm start`
